# AquaCommandAI 🌊🤖

> AI-powered Water Resource Intelligence Platform for crop recommendation, yield prediction, irrigation forecasting, reservoir risk analysis, and agricultural decision support.

## 📖 Overview

AquaCommandAI is a full-stack AI-powered platform developed to support smart agriculture and sustainable water resource management. The system integrates multiple Machine Learning models to provide crop recommendations, crop yield prediction, water demand forecasting, reservoir risk assessment, and intelligent advisory services.

The platform combines data-driven insights with modern web technologies to assist farmers, researchers, and water resource managers in making informed decisions.

---

## 🚀 Features

### 🌱 Crop Recommendation
Recommends the most suitable crop based on soil nutrients and environmental conditions.

### 📈 Crop Yield Prediction
Predicts crop yield using agricultural and environmental parameters.

### 💧 Water Demand Forecasting
Estimates irrigation water requirements for efficient water management.

### 🏞 Reservoir Risk Monitoring
Analyzes reservoir water levels and identifies potential risk conditions.

### 🤖 AI Advisory System
Provides intelligent recommendations to support agricultural planning and resource allocation.

### 📊 Interactive Dashboard
Displays predictions and analytics through a user-friendly dashboard.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Devakumar-A/AquaCommandAI.git
cd AquaCommandAI
```

### Backend Setup

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn app:app --reload
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
